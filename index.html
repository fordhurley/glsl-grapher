What?

This is a visualization tool for GLSL shader functions, using WebGL. It will graph any function with the signature float y(float x).

Why?

When writing shader code, it can be hard to visualize the output of a given function. There are a million other graphing tools, but all require you to translate your shader code into another format. This allows you to graph the output of your actual code.

Examples?

  • Step
  • Smoothstep
  • Multi-smoothstep
  • Gradient noise
